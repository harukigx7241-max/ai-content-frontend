# アンチパターン集 — やってはいけないプロンプト・コンテンツパターン

> 適用範囲: 全カテゴリ共通  
> 最終更新: 2026-04-16

---

## 1. プロンプト設計のアンチパターン

### ❌ 役割定義なしのプロンプト
```
悪い例: 「副業の記事を書いてください」
良い例: 「あなたはnoteで月収50万円を達成したプロコンテンツクリエイターです。副業初心者向けの有料記事を書いてください」
```
**理由:** 役割定義がないと AI の回答品質が安定せず、表面的な内容になりがち

### ❌ 抽象的すぎる指示
```
悪い例: 「良い記事を書いてください」
良い例: 「2000字、ですます調、25〜35歳の会社員向け、冒頭にQ形式のフックを置いた記事を書いてください」
```

### ❌ 複数の相反する指示
```
悪い例: 「簡潔に、でも詳しく、プロ向けで初心者にもわかりやすく」
良い例: ターゲットを一つに絞り、そのターゲットの言語レベルに合わせる
```

### ❌ 出力形式未指定
```
悪い例: 記事を書いて
良い例: 以下の形式で出力: [タイトル] [リード文] [H2見出し × 5] [本文 各見出し500字] [まとめ]
```

## 2. コンテンツ設計のアンチパターン

### ❌ タイトルと内容のミスマッチ
「月収100万円の方法」というタイトルで、「努力が大切」という内容

### ❌ 具体性の欠如
- 数字がない（「多く」「たくさん」）
- 事例がない（「できた人がいます」のみ）
- 手順がない（「やってみよう！」で終わる）

### ❌ 読者の悩みより著者の主張が優先
読者が「どうすれば稼げるか」を知りたいのに、著者の自慢話が8割

### ❌ 有料壁の位置が不適切
無料部分が短すぎると購入動機が生まれない。最低でも全体の30%を無料で提供する

## 3. SNS特有のアンチパターン

### ❌ 宣伝色が強すぎる
「今すぐ買ってください！！！」系の投稿は即ミュート・フォロー解除の原因

### ❌ 長すぎる最初の1文
XではFirst ViewのInpression（最初の数文字）が命。最初の1文は30字以内を目安に

### ❌ ハッシュタグ乱用
#副業 #在宅 #フリーランス ... を30個つけても逆効果。関連性の高い3〜5個が最適

---

## 4. 占い・スピリチュアルコンテンツのアンチパターン

### ❌ 断言しすぎる表現
「必ず○○になります」「絶対に上手くいきます」は信頼を損なう

### ❌ 個人情報・感情に踏み込みすぎる
鑑定書でクライアントのプライバシーや心理的傷を不用意に指摘しない

### ❌ 金額保証・具体的予言
「この鑑定を受けると月収が上がります」等の利益保証は景品表示法に抵触する可能性
